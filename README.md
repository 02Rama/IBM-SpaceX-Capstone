# SpaceX Falcon 9 Landing Prediction

## IBM Applied Data Science Capstone Project

### Project Overview

The goal of this project is to predict the success of SpaceX Falcon 9 first-stage landings using historical launch data.

### Project Tasks

* Data Collection using API and Web Scraping
* Data Wrangling and Cleaning
* Exploratory Data Analysis (EDA)
* SQL Analysis
* Interactive Visual Analytics with Folium
* Dashboard Development with Plotly Dash
* Predictive Analysis using Machine Learning

### Repository Structure

* notebooks/ : Jupyter notebooks
* data/ : datasets
* dashboard/ : Dash application files
* images/ : screenshots and visualizations

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Folium
* Scikit-learn
* SQL
